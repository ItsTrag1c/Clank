# Code of Conduct

## Our Pledge

We're building AI tools for everyone. This project is open to all contributors regardless of background, experience level, or identity. We pledge to make participation a respectful and constructive experience.

## Expected Behavior

- Be kind and welcoming to newcomers
- Give constructive, specific feedback
- Respect differing viewpoints and experiences
- Focus on what's best for the project and its users
- Accept responsibility for mistakes and learn from them

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information without consent
- Sustained disruption of discussions or contributions
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers will remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct. Repeated or serious violations will result in a ban from the project.
